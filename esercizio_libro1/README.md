Esercizi HTML asincroni

Questa cartella contiene 4 esercizi (HTML + sorgenti) che dimostrano il caricamento asincrono di documenti HTML mediante mouse hover e menu di selezione.

File principali:
- exercise1.html: due stringhe su cui posizionare il mouse per caricare due documenti diversi in due zone.
- exercise2.html: menu a 3 opzioni che carica 3 documenti asincroni nei tre <div>.
- exercise3.html: due liste di 4 elementi; hover su ogni elemento carica documenti distinti in due zone.
- exercise4.html: menu a 6 opzioni che carica 6 documenti in tre <span> secondo la mappatura richiesta.

Apri i file HTML nel browser (o con "Apri con Live Server") per provare le funzionalità.
